# belajar LAN dasar

menggunakan 4 end device (komputer) yang dihubungkan dengan menggunakan switch, kemudian dilakukan pengalamatan dengan alamat ip

1. komputer 0 : 192.168.1.1
2. komputer 1 : 192.168.1.2
3. komputer 2 : 192.168.1.3
4. komputer 3 : 192.168.1.4

percobaan pengalamatan menggunakan ping dan terbukti berhasil dan pengalamatan
