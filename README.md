# belajar_router_dasar

Router pada cisco packet tracer dapat diaktifkan dengan menggunakan mode CLI dan fisik
Mode CLI memiliki keuntungan lebih karena dapat menjangkau semua perintah pada router

router CLI memiliki beberapa mode, yaitu
1. user exec mode (tanda >) mode default
2. privilige exec mode (tanda #) menggunakan perintah enable
3. global config mode (tanda (conf)#) menggunakan perintah configure terminal
   dapat digunakan untuk memberi nama host router dengan menggunakan perintah hostname diikuti dengan nama

Konfigurasi password pada mode 1 dan 2
